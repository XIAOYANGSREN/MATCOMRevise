# Submitted result artifacts

The manuscript was prepared from representative submitted runs. Lightweight text outputs are mirrored in this repository when practical.

Binary artifacts such as model weights (`.pth`), NumPy archives (`.npz`), and pre-rendered figure files are included in the supplementary archive submitted with the manuscript and will be included in the final DOI archive after acceptance.

The scripts in `03_code/` and `04_new_experiments/` can regenerate these artifacts when the full data package and Python environment are available.
